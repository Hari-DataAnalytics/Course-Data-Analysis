
📊 Online Courses Enrollment Dashboard

📌 Overview

The Online Courses Enrollment Dashboard provides insights into course performance, revenue generation, and student enrollments across different categories and states. This dashboard helps stakeholders make data-driven decisions by visualizing key metrics in an interactive way.

🚀 Features
📂 Category Filter – Filter courses by Business, Data, Finance, and Technology

🌍 State Filter – Analyze performance across multiple states

📚 Course Filter – View metrics for specific courses

📅 Date Filter – Track trends over time

💰 Total Revenue – Displays overall revenue generated

👨‍🎓 Total Enrollments – Shows number of students enrolled

📖 Total Courses – Count of available courses

📊 Revenue by Course – Compare course-wise revenue

📍 Revenue by State – Analyze geographic performance

🛠️ Tools & Technologies Used

Power BI / Excel (Dashboard Creation)

Data Cleaning & Transformation

Data Visualization Techniques

📈 Key Insights
Alteryx generates the highest revenue among all courses
Florida leads in revenue among all states
Technology-related courses dominate overall enrollments
Revenue distribution varies significantly by location
